## Description

The Temperature Monitoring System is an embedded systems project developed using Arduino Uno and a temperature sensor. The objective of this project is to measure the surrounding temperature and display the readings on the Serial Monitor in real time.

The temperature sensor continuously senses the ambient temperature and sends the analog data to the Arduino microcontroller. The Arduino processes this data, converts it into temperature values in degrees Celsius, and displays the results on the Serial Monitor.

This project demonstrates the interfacing of sensors with microcontrollers and highlights the importance of real-time data acquisition in embedded systems. Temperature monitoring is widely used in industrial automation, environmental monitoring, healthcare equipment, and smart home applications.

The system provides accurate and continuous temperature measurements, making it suitable for various monitoring applications. The project was successfully implemented and tested using Arduino simulation tools.
